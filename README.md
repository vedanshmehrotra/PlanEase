   # FS Project 2025
   
   A full-stack project for task and note management.
   
   ## Features
   - User authentication
   - Task management
   - Note management
   - Dashboard
   
   ## Tech Stack
   - Frontend: HTML, CSS, JavaScript
   - Backend: Node.js, Express
   - Database: MongoDB
   
   ## Setup
   1. Clone the repository
   2. Install dependencies: `npm install`
   3. Start the server: `node backend/server.js`
